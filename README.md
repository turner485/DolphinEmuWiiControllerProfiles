# Wii Profiles

This repo contains custom Dolphin Wiimote profiles for individual Wii games.
**PLEASE NOTE**
These profiles are not yet perfect and are still undergoing minor twaeks.

## Setting up per game automatic profiles
To configure this right click the game > prperties > game config > select the editor tab > enter the below in the User Config.

```
[Control]
Wiimote Source0 = 0
WiimoteProfile1 = NewSuperMarioBrosWii
```

## Profiles in this repo

- `Bully.ini`
- `CastlevaniaJudgment.ini`
- `GrillOffwithUltraHand.ini`
- `MarioKartWii.ini`
- `Muscle.ini`
- `NewSuperMarioBrosWii.ini`
- `TatsunokovsCapcomUltimateAll-Stars.ini`

## Notes

- These profiles are built for `XInput/0/Gamepad`.
- Some games use Wii Remote only, while others use `Nunchuk` or `Classic` extension mappings.
- If a game feels wrong, the profile can be adjusted per game without affecting the others.
